# Newspaper OCR — article segmentation for scanned newspaper pages

Takes a scanned newspaper page image and produces structured JSON: individual article
regions (bounding boxes) with per-region OCR text — instead of one flat blob of page OCR.
Built for search, indexing, and historical text analysis of newspaper archives.

## Setup

Requires Python 3.13+ and a [Tesseract](https://github.com/tesseract-ocr/tesseract)
install on PATH (developed against Tesseract 5.5.0).

```bash
pip install -r requirements.txt
```

## Usage

```bash
python -m newspaper_ocr.run --input page.jpg --output result.json
# optional: --debug-dir out/   saves intermediate images (binarized page, region overlays)
# optional: --config my.yaml   alternative parameter set (default: repo-root config.yaml)
```

Until all phases are implemented, the CLI runs the implemented prefix of the pipeline and
exits with a message at the first unimplemented stage.

```json
{

  "page_size": [1220, 1490],
  "regions": [
    {

      "ocr_confidence": 88.5,
      "region_type": "unclassified"
    }
  ]
}
```

the real-page check used by the verification protocol runs against
